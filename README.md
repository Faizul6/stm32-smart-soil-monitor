# STM32 Smart Soil Monitoring System

Embedded system project using STM32 NUCLEO-F401RE for:

- soil moisture sensing
- temperature and humidity sensing with DHT11
- LED status indication
- 16x2 LCD display output

## Hardware
- STM32 NUCLEO-F401RE
- Soil moisture sensor
- DHT11
- 16x2 LCD
- Red, Yellow, Green LEDs

## Features
- Reads moisture using ADC
- Reads temperature and humidity
- Shows values on LCD
- Indicates soil status with LEDs
